## 💡 Key Takeaway
- 环境变量读取语法
  - 使用  $变量名  读取变量值，搭配  echo  打印输出，例： echo $PATH 。

- 两条查看变量命令区别
  - env ：仅输出系统环境变量（全局变量，所有Shell会话可用）
  - set ：输出全部变量，包含系统环境变量 + 自定义局部变量 + Shell函数

- 系统环境变量特性
  - 系统开机自动初始化，全局生效；
  - 作用：存储系统路径、用户信息、主机信息、语言配置等；
  - 子Shell、脚本均可直接调用。

- 变量命名规范
  - 系统内置变量统一大写，自定义变量推荐小写区分。

## ⚙️ Practice
```bash
# 查看当前Shell系统环境变量
env

# 查看Shell全部变量(系统环境变量+自定义变量+函数)
set

# 命令的搜索路径，多路径以冒号：分割
echo $PATH

# 显示当前用户执行命令的历史列表文件
echo $HISTFLE

# 设置当前系统语言环境
echo $LANG

# 当前用户主目录
echo $HOME

# 当前shell解析器类型
echo $SHELL

# 显示当前所在路径
echo $PWD

# 显示之前的路径
echo $OLDPWD

# 显示当前主机名
echo $HOSTNAME

# 显示主机的架构
echo $HOSTTYPE
