## 💡 Key Takeaway
-  连通测试： ping （ -c  指定发包数量）
- 文件/接口请求： curl （ -I  仅看响应头、 -O  下载文件）、 wget 
- 端口连接查看： ss -tulnp 、 netstat -tulnp （依赖net-tools）
- 网卡信息： ip addr show 、 ifconfig

🛠️ Troubleshooting
 
- 遇到的问题：
  - curl -O  /  wget  报错  no URL specified 
  - netstat: command not found 
  - scp: stat local xxx: No such file or directory 
  - SSH首次连接提示主机指纹确认

- 报错原因：
  - 参数后未跟完整下载链接，URL与命令粘连/中间带空格
  - 新版EL9系统默认未预装net-tools软件包
  - 本地当前目录不存在待传输文件/文件夹
  - 本地无远程服务器身份记录，安全校验机制

- 解决办法：
  - 命令、参数、URL之间加空格，完整链接不拆分空格
  - 执行 yum install net-tools -y 安装，或直接使用替代命令 ss -tulnp 
  - touch 创建文件、 mkdir 创建目录后再执行传输；传文件夹必须加 -r 参数
  - 输入 yes 确认信任，输入远程用户密码完成公钥推送
