## 💡 Key Takeaway
- grep文本过滤:-i 忽略大小写、 -v 反向、 -r 递归、 -n 行号、 -o 仅匹配片段、 -E 扩展正则； -A/-B/-C 查看上下文
- sed流编辑： s/old/new/g 全局预览替换， -i 就地修改； 数字d 删除行、 -n 数字p 打印指定行、 /start/,/end/d 区间删行
- awk列处理： $1/$NF 提取列， -F 自定义分隔符；数值条件过滤、 sum 累加求和、 NR 统计总行数
- tar打包压缩： c 创建、 x 解压、 v 详情、 f 文件（放最后）； z gzip(.tar.gz)、 j bzip2(.tar.bz2)；gzip/gunzip单文件压缩解压，zip/unzip处理zip格式
- 正则符号： [] 匹配单个字符集、 {} 限定匹配次数、 + 匹配1次及以上，扩展正则必须加 -E 
- 工具串联：管道 | 传递输出，多工具链式处理文本

## ⚙️ Practice
```bash
#前置环境初始化
mkdir -p test_dir/sub1 test_dir/sub2
cat > data.txt <<EOF
User admin Age 25 Score 88
user test Age 18 Score 59
USER ROOT Age 30 Score 95
user guest Age 22 Score 72
#
user dev Age 28 Score 45
EOF
cp data.txt test_dir/sub1/
cp data.txt test_dir/sub2/
cp data.txt config.txt
cat > score.txt <<EOF
10
20
30
40
50
EOF

#grep过滤练习
##1.忽略大小写匹配user，同时显示行号
grep -in "user" data.txt
##2.反向匹配，过滤掉包含user的所有行
grep -v "user" data.txt
##3.递归搜索test_dir目录内所有含Score的文件并显示行号
grep -rn "Score" test_dir/
##4.仅提取文件内所有连续数字串
grep -oE "[0-9]+" data.txt
##5.扩展正则匹配两位数，附带匹配行后2行内容
grep -E -A 2 "[0-9]{2}" data.txt
##6.匹配admin所在行，查看匹配行前1行
grep -B 1 "admin" data.txt
##7.匹配ROOT所在行，查看匹配行前后各2行上下文
grep -C 2 "ROOT" data.txt

#sed编辑练习
##1.预览全局替换user为staff，不修改原文件
sed 's/user/staff/g' config.txt
##2.就地修改文件，将所有admin替换为superadmin
sed -i 's/admin/superadmin/g' config.txt
##3.仅打印文件第3行，屏蔽其余行输出
sed -n '3p' config.txt
##4.直接修改文件，删除2~4行内容
sed -i '2,4d' config.txt
##5.删除#注释行到空行之间所有内容
sed -i '/^#/,/^$/d' config.txt

#awk列处理练习
##1.打印每行第1列、最后一列
awk '{print $1,$NF}' data.txt
##2.将空格替换为冒号分隔，提取年龄数字列
sed 's/ /:/g' data.txt | awk -F: '{print $4}'              = awk -F: '{print $3}' <(sed 's/ /:/g' data.txt)
##3.条件过滤，只输出分数大于60的完整行
awk '$NF > 60' data.txt
##4.累加score.txt第一列所有数字，输出总和
awk '{sum+=$1} END{print sum}' score.txt
##5.过滤注释行后，统计有效数据总行数
grep -v "^#" data.txt | awk 'END{print NR}'

#打包压缩练习
##1.tar仅打包不压缩
tar -cvf all.tar test_dir/
tar -xvf all.tar
##2.tar.gz gzip压缩包
tar -czvf test.tar.gz test_dir/
tar -xzvf test.tar.gz
##3.安装bzip2工具后创建tar.bz2压缩包
yum install -y bzip2
tar -cjvf cfg.tar.bz2 config.txt
tar -xjvf cfg.tar.bz2
##4.gzip单文件压缩、解压
gzip score.txt
gunzip score.txt.gz
##5.安装zip/unzip工具，处理zip格式
yum install -y zip unzip
zip data.zip data.txt
unzip -o data.zip

#综合管道串联实操
grep -rin "user" test_dir/ | grep -v "^#" | sed 's/user/worker/g' | awk '$NF >=70 {print $1,$NF}' > final_result.txt
tar -czvf result.tar.gz final_result.txt
 
🛠️ Troubleshooting
 
- 遇到的问题：
  -  grep -o "[0-9]+" data.txt 无输出
  - 进程替换  < (sed ...)  报语法错误 
  - sed执行替换空格命令报 unterminated s' command 
  - tar -cjvf 提示 bzip2: command not found 
  - zip 命令提示command not found
  - unzip解压弹出文件覆盖询问

- 报错原因：
  - 普通grep不识别 + 扩展正则符号， + 被当做普通文本字符匹配
  - < 与 ( 中间存在多余空格，破坏shell进程替换语法
  - 空格拆分单引号，替换表达式未完整闭合
  - 系统未预装bzip2压缩工具
  - 系统缺少zip/unzip工具包
  - 当前目录存在同名文件，工具防止误删除数据

- 解决办法：
  - grep添加 -E 参数启用扩展正则： grep -oE "[0-9]+" data.txt 
  - 去掉 < 与 ( 中间空格 <(sed 's/ /:/g' data.txt) ，或改用管道 | 传递数据
  - 完整使用单引号包裹替换表达式： sed 's/ /:/g' data.txt 
  - 执行 yum install -y bzip2 安装工具，或改用 z 参数生成tar.gz包
  - 执行 yum install -y zip unzip 安装zip工具
  - 输入 y 单文件覆盖；或使用 unzip -o data.zip 强制无交互覆盖
